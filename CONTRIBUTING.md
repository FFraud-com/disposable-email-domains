# Contributing

This list grows with the community.

## Submit a missing domain

- **Fastest:** report it at [ffraud.com/report](https://ffraud.com/report).
- **Or open an issue** here with the domain(s) and, ideally, a link to the disposable-mail service that operates them.

Confirmed domains ship in the next daily build.

## Dispute a listing

Operate a legitimate mail domain that landed here? Open an issue and we recheck on the next build.
